# 提示词优化指南

> **目标**：提升AI输出质量  
> **适用**：所有使用AIDE标准的AI员工

---

## 提示词公式

### 基础公式

```
[角色] + [任务] + [输入] + [输出要求] + [约束条件]
```

### 示例

```
角色：你是专业的短视频创作助手
任务：根据选题生成即梦Seedance视频提示词
输入：选题"武林高手面试记"
输出要求：
  - 包含：主体+动作+场景+风格+镜头+特效
  - 中英双语
  - 电影级质感
约束条件：
  - 不要出现暴力内容
  - 适合全年龄段观看
```

## 优化技巧

### 1. 具体化

❌ 差：
```
生成一个视频提示词
```

✅ 好：
```
生成一个15秒竖屏视频提示词，主题是"武侠高手穿越现代职场"，要有喜剧效果，适合抖音平台
```

### 2. 结构化

使用清晰的结构：
```
## 角色
你是...

## 任务
请...

## 输入
数据：...

## 输出格式
1. ...
2. ...

## 约束
- ...
- ...
```

### 3. 示例化

提供示例（Few-shot）：
```
请参考以下示例生成：

示例1：
输入：修仙者挤地铁
输出：一位古装修仙者，御剑飞行在拥挤的地铁车厢中，乘客们惊讶地看着他，现代与古代反差，搞笑风格，广角镜头，电影级光影

示例2：
输入：剑客送外卖
输出：...

现在请生成：
输入：[用户输入]
输出：
```

## 常见错误

### 错误1：太抽象
- ❌ "做一个好看的视频"
- ✅ "一位身穿白袍的女侠客，在咖啡厅里用剑气切蛋糕，顾客震惊的表情，暖色调光影，慢动作特写，电影级质感"

### 错误2：太复杂
- ❌ 一次性要求10个不同动作
- ✅ 一次只要求1-2个核心动作

### 错误3：缺少约束
- ❌ 没有说明不能做什么
- ✅ 明确列出禁止事项

## AIDE专用Prompt模板

### RACI-D分配Prompt

```
你是AI员工设计顾问。

任务：帮助用户分配RACI-D职责

输入：用户描述的任务

输出格式：
R（人类执行）：...
A（人类负责）：...
C（AI咨询）：...
I（AI知情）：...
D（AI委托）：...

分配原则：
1. A必须是人类
2. 关键决策点必须有A
3. AI做重复性任务（D/I）
4. 人类做创造性决策（R/A）
```

### STATE设计Prompt

```
你是工作流设计专家。

任务：根据任务设计STATE状态机

输入：任务清单

输出格式：
[IDLE] 触发条件
  ↓
[AUTO] AI自动执行内容
  ↓
[WAIT] 暂停等待人类决策
...

设计原则：
1. 每个WAIT有超时机制
2. 关键节点有CHECK
3. 必须人工的节点用HALT
```
